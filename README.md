# Hệ Thống Quản Lý Chấm Điểm Báo Cáo - Frontend

Dự án Frontend xây dựng bằng React, TypeScript, Vite và TailwindCSS.

## 🚀 Hướng Dẫn Cài Đặt & Khởi Chạy Dự Án

### Yêu cầu hệ thống:
- **Node.js**: Phiên bản 18 trở lên.

### Bước 1: Cài đặt thư viện
Mở terminal tại thư mục `CNPM-frontend` và chạy lệnh sau:
```bash
npm install
```

### Bước 2: Khởi chạy giao diện
Sau khi quá trình cài đặt hoàn tất, hãy khởi động local server bằng lệnh:
```bash
npm run dev
```

Server sẽ tự động cấp một địa chỉ mạng cục bộ (thường là `http://localhost:5173`). Bạn có thể nhấn giữ phím Ctrl/Cmd và click vào đường link hiện trong terminal để mở giao diện trên trình duyệt web.

---

### 🛠 Các lệnh hỗ trợ khác
- `npm run build`: Build dự án ra sản phẩm chuẩn bị deploy (sẽ tạo thư mục `dist`).
- `npm run preview`: Xem trước bản build trên trình duyệt.
- `npm run lint`: Chạy trình kiểm tra và chuẩn hóa code bằng ESLint.
